Como Usar o QuickBoard

Fácil de começar mesmo para usuários iniciantes

1

Abrir o Quadro Tático

Simplesmente acesse o QuickBoard no seu navegador. Nenhuma instalação necessária.

2

Posicionar Jogadores

Clique duas vezes (toque) na área vazia do campo para adicionar jogadores. Arraste-os para as posições desejadas.

3

Personalizar Jogadores

Clique duas vezes (toque) nos jogadores para definir seus números, nomes e cores.

4

Compartilhar

Compartilhe seu quadro tático finalizado com companheiros de equipe via URL. Fácil de compartilhar nas redes sociais também.
